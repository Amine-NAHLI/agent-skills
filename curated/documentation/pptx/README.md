## Why selected

- **Production reference** — powers Claude's presentation capabilities
- **Full lifecycle** — create from scratch, edit existing decks, extract and reorganize content
- **Rich tooling** — bundled Python scripts, OOXML validators, and theme helpers
- **Common use case** — pitch decks, slide presentations, and template work are frequent agent tasks
- **Official Anthropic skill** — authoritative patterns for presentation document skills
