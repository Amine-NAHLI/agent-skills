## Why selected

- **Universal document format** — PDF tasks are among the most common non-code agent requests
- **Production-grade** — same skill family used in Claude's document features
- **Complete coverage** — from simple extraction to forms, encryption, and OCR
- **Bundled scripts** — practical Python helpers agents can invoke directly
- **Official Anthropic reference** — authoritative patterns for PDF manipulation
