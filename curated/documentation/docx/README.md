## Purpose

Create, read, edit, and manipulate Word documents (`.docx`, `.dotx`) — formatting, tables of contents, tracked changes, comments, templates, and professional document generation with bundled Python tooling.

## Why selected

- **Production reference implementation** — powers Claude's document capabilities; one of the most complete document skills available
- **Rich tooling** — includes validation scripts, OOXML schemas, and office helpers
- **High reuse value** — reports, memos, letters, and templates are common agent tasks
- **Well documented** — extensive SKILL.md with clear trigger conditions
- **Source-available reference** — Anthropic shares this as a model for complex production skills
