## Purpose

Guides agents to produce distinctive, intentional frontend UI — aesthetic direction, typography, layout, motion, and design choices that avoid generic templated "AI slop" defaults.

## Why selected

- **High-impact frontend skill** — one of the most referenced design skills in the agent ecosystem
- **General-purpose** — applies to React, Vue, HTML/CSS, and any web UI work in Cursor, Claude Code, or Codex
- **Opinionated and actionable** — concrete principles agents can follow, not vague "make it pretty" advice
- **Complements vercel-react-best-practices** — design quality paired with React performance optimization
- **Official Anthropic skill** — actively maintained, popular on [skills.sh](https://skills.sh/anthropics/skills/frontend-design)
