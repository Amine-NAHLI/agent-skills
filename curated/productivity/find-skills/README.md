## Purpose

Helps users discover and install agent skills from the open ecosystem — search via the Skills CLI (`npx skills`), evaluate install counts and source reputation, and install skills from trusted publishers.

## Why selected

- **Meta-productivity skill** — essential for navigating the growing agent skills ecosystem
- **Pairs with this collection** — teaches agents how to find and evaluate skills like those curated here
- **Official Vercel skill** — maintained by the team behind the Skills CLI and [skills.sh](https://skills.sh/)
- **General-purpose** — triggers on "how do I do X", "find a skill for…", and capability extension requests
- **Ecosystem infrastructure** — documents the standard install workflow used across Cursor, Claude Code, and Codex
